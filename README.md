# 一个用于选择和测试不同npm镜像的命名行工具

## 安装
``` bash
npm i mirror-switching -g
```

## 使用方法
### 列出所有的可用的镜像
``` bash
zmp ls
```
### 选择镜像
``` bash
zmp use
```
### 查看当前使用的镜像
``` bash
zmp current
```
### 测试镜像速度
``` bash
zmp ping
```

如果您在使用过程中遇到任何问题或者有建议，请随时在 [GitHub](https://github.com/Zephyr-Zhang99/mirror-switching) 上提出。
